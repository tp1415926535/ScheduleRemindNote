版本v1.0
最近更新：2020.5.5

  显示：
    中心：当前日程事件
    标题：事件与时间
    左侧：当前时间于一天位置
    下方：事件完成度
    右上：置顶与否
    总览窗体：显示全部日程事件
    编辑窗体：显示编辑栏用于自定义日程，可本地保存

  功能：
    滚轮：调节透明度
    双击：显示事件总览窗体
    右键：显示日程编辑窗体
    切换：日程事件按时间自动切换，有提示音
